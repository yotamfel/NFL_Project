# פלטפורמת נתוני NFL — סיכום הפרויקט

---

## מה הפלטפורמה עושה

פלטפורמת ווב אינטראקטיבית שמנגישה נתוני NFL לכל משתמש — ללא ידע טכני. המשתמש יכול לחפש שחקנים, לצפות בפרופיל מלא, להשוות שחקנים, לנתח ערך דראפט, לחקור מגמות ליגה, ולשאול שאלות בשפה חופשית.

**פרוסה בכתובת:** https://nfl-data-platform-production.up.railway.app/

---

## ארכיטקטורה טכנית

| שכבה | טכנולוגיה |
|---|---|
| בסיס נתונים | PostgreSQL 17 (Neon, serverless cloud) |
| צד שרת | Python + FastAPI + SQLAlchemy 2.0 |
| מודל ML | scikit-learn HistGradientBoostingRegressor |
| חיפוש חכם | Claude Haiku (Anthropic API) |

| פריסה | Railway — מכולת Docker יחידה (FastAPI משרת גם את ה-API וגם את ה-React) |

## מודולים / דפים

### 1. דף שחקן (Player Profile)
- חיפוש שחקן לפי שם עם פילטרים (עמדה, עונה, קבוצה)
- כרטיס כותרת: עמדה, שנות פעילות, נתוני דראפט, מדדי קומביין
- טבלאות סטטיסטיקה עונה-אחר-עונה עם מצב Basic ו-Advanced
- גרפי קריירה עם אפשרות לבחור סטטיסטיקה מהתפריט (Selectable Charts)
- פסים אדומים בגרפים לעונות שבהן השחקן החסיר 4+ משחקים
- **היסטוריית פציעות (2009+):** Out/Doubtful/Questionable מדוחות שבועיים + משחקים חסרים מוערכים מנתוני G (מזהה שחקני IR)
- **Snap Counts (2013+):** אחוז נוכחות עונתי ושבועי
- **Advanced Receiving (WR/TE/RB):** ADOT, YAC, BrkTkl, Drop%, TgtRtg, Sep, Cush, YAC+ (PFR 2018+, NGS 2016+)
- **Next Gen Stats — QB (2016+):** TT, IAY, CAY, ADOTS, Aggr%, CPOE, MaxDist
- **Next Gen Stats — RB (2016+):** Eff, TLOS, RYOE/A, RPOE%, 8-Box%
- שמירת שחקן עם כוכב ★ + הוספת הערה אישית

### 2. השוואת שחקנים (Comparison)
- השוואת עד 4 שחקנים במקביל
- מיתוג קריירה מלאה מול עונה בודדת
- כל קטגוריות הסטטיסטיקה: Passing, Offense, Defense, Kicking, Punting, Returns
- Leaderboard — 20 השחקנים המובילים בכל Stat, עם הדגשה של השחקנים שנבחרו
- שמירת השוואה + הערה

### 3. ניתוח דראפט (Draft Analysis)
- גלישת כל בחירות הדראפט 2000–2025 עם פילטרים
- Career AV כמדד איכות קריירה ניטרלי-עמדה
- Steals: שחקנים מסיבוב 4+ עם Career AV גבוה (ערך מופתע)
- Busts: שחקנים מסיבובים 1–2 עם Career AV נמוך
- המלצות ML: תחזית Career AV לפי נתוני קומביין, מציאת הפערים
- Round Stats: ממוצע וחציון לפי סיבוב
- Custom Query: בנה פילטר מותאם אישית

### 4. חיפוש חכם (Smart Search)
- שאלה חופשית בעברית או אנגלית
- Claude מתרגם ל-SQL ומריץ מול בסיס הנתונים
- תצוגת תוצאות + צפייה ב-SQL שנוצר
- שמירת חיפוש + הערה

### 5. מגמות ליגה (League Trends)
- גרף מגמה עונתית לכל Stat בכל קטגוריה
- פילטר לפי עמדה, טווח שנים, Sum/Average
- גרף קווסטות לפי קבוצה (צבעי NFL רשמיים)
- טבלת פירוט עונה-אחר-עונה

### 6. Saved — פריטים שמורים
- שמירת שחקנים, השוואות וחיפושים
- הוספת הערה אישית לכל פריט שמור
- טאב Notes: לוח כתיבה חופשי עם עריכה inline
- כל הנתונים בלוקאלסטורג' — לפי שם משתמש

### 7. מדריך (Guide)
- מדריך משתמש מלא בתוך האפליקציה
- שתי שפות: עברית ואנגלית (ניתן לעבור ביניהן)
- מכסה את כל הפיצ'רים כולל הגדרות סטטיסטיקות
